Why these packages are included?

| Package                 | Purpose                             |
| ----------------------- | ----------------------------------- |
| `networkx`              | digital thread graph construction   |
| `pydantic`              | structured artifact / claim models  |
| `numpy`                 | numeric drift scoring               |
| `scikit-learn`          | cosine similarity utilities         |
| `sentence-transformers` | embedding-based semantic similarity |
| `transformers`          | backend for embedding models        |
| `torch`                 | model runtime                       |
| `tqdm`                  | progress bars                       |
| `pyyaml`                | loading semantic constraint files   |
| `typer`                 | CLI for pipeline scripts            |
| `rich`                  | readable console output             |

Optional (lightweight mode)

If you want the artifact to run without downloading large models (sometimes preferred for ACM artifact review), you can instead use this minimal version:

networkx>=3.2
pydantic>=2.6
numpy>=1.26
scikit-learn>=1.4
pyyaml>=6.0
tqdm>=4.66

Then embeddings can be optional.
